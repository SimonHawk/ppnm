The exponential function can be defined as a solution to the differential equation:
\begin{equation}\label{diff-eq}
y'(x) = y(x) \;,
\end{equation}
with boundry condition:
\begin{equation}\label{bound-cond}
y(0) = 1 \;.
\end{equation}
